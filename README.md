# 2018, PNU CSE, Web Programming : web quiz

# features

1. 문제 작성후 서버에 저장/확인가능
2. 문제 풀기 기능으로 서버의 문제를 불러온후 응시가능. 문제는 무작위로 선택되며 객관식(단일/복수정답),주관식으로 구성
3. 시험 결과를 서버에 저장/확인가능
